About the Project

Student Profile

This activity introduces students to the complete deployment process of taking a simple website from local files to a public URL on the Internet. It combines web development basics with foundational cloud computing concepts.

Students will create a website with a home page, an about page, and a contact page, then host it using a free platform and verify that the site is accessible online.

Project Summary

Students will learn how to build a simple multi-page website and publish it on a free cloud hosting service such as Netlify, Render, or GitHub Pages.

What Students Will Learn

  • How to build a basic multi-page HTML site
  • How to style content using CSS
  • How to publish a website to the cloud
  • How to test and confirm a public web address

Project Procedures

1. Create an account on the selected hosting platform.
2. Develop a simple website with the required pages and content.
3. Test the site locally to make sure all links and pages work.
4. Upload or connect the files and publish the website.
5. Verify the public URL and capture screenshots for submission.
